# ACME Robotics FY2024 — Synthetic Demo Dataset

**Everything here is fictional and machine-generated** for teaching multi-modal RAG.
"ACME Robotics", the figures, the documents, and the video/transcript do not describe any
real company, person, or product. No third-party or copyrighted material is included.

## Contents
- `images/`  six original charts (PNG) — the numeric facts live in the pixels.
- `text/`    six short Markdown documents.
- `video/fy2024_review.mp4`  a ~45s narrated-slide video (silent; visuals only).
- `video/fy2024_review_transcript.json`  timestamped transcript (as if produced by ASR).
- `video/keyframes/`  reference keyframes (one per slide) for environments without OpenCV.
- `data_dictionary.json`  what each artifact contains + a demo question it answers.
