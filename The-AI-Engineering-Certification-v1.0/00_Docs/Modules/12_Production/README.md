# Session 12: 🛤️ Production 101: Guardrails & Caching

🎯 Learn a few upgrades: for performance and security/trustworthiness.

📚 **Learning Outcomes**

- Understand guardrails, including the key categories of guardrails
- Understand the importance of semantic caching
- How to use Prompt and Embedding caches

No new tools or accounts are required

## 📜 Recommended Reading

This week's focus remains on the Certification Challenge, so instead of introducing another major topic, we'll cover several production-oriented concepts that don't quite warrant a full session but are valuable as you begin building real-world AI applications.

We'll dive into **guardrails** and **caching**, two of the highest-impact, lowest-effort improvements you can make to an AI system. We'll also provide exploratory notebooks on **security**, **Agent-to-Agent (A2A) communication**, and **observability & monitoring** so you're familiar with the broader production landscape and know where to go next as your applications mature.
