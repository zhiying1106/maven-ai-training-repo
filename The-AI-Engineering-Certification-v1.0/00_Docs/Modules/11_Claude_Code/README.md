# Certification Challenge

Introducing coding assistant best practices is all about preparing you to succeed on the Certification Challenge. This challenge brings together the most important concepts you've learned so far and asks you to apply them to a real AI engineering problem. More details and the full task list will be shared soon.

For now, start thinking about a use case you'd like to build for the remainder of the cohort. Tomorrow, you'll have dedicated class time to discuss your ideas with your journey group and refine them before starting the challenge.

### Choosing a Great Use Case

A strong Certification Challenge project should:
- Be something you're genuinely interested in, whether it's related to your work, hobbies, or another area you know well.
- Solve a problem that can't be adequately handled by simply prompting ChatGPT. Your application should provide clear additional value through architecture, tools, agents, RAG, workflows, or other AI engineering techniques.
- Be challenging enough to leverage the concepts we've covered throughout the cohort. If the solution is just "call an LLM once," it's probably too simple.

Aim for a project where thoughtful engineering decisions matter just as much as the prompt itself.
